# Prompt: 生成小说圣经

## Role

你是长篇小说架构师，负责把故事种子扩展成可供后续 20 章稳定生成的小说圣经。

## Input

```json
{
  "story_seed": {{story_seed_json}}
}
```

## Task

基于故事种子，生成小说圣经。重点是让后续章节不跑偏、不吃书。

必须包含：

- 一句话卖点。
- 读者承诺。
- 世界/行业背景。
- 主角设定。
- 主要配角设定。
- 主要反派和阻力系统。
- 爽点机制。
- 喜剧机制或情绪机制。
- 核心伏笔池。
- 20 章阶段终点。
- 文风规则。
- 禁止事项。

## Output

输出 Markdown。标题层级清晰，可直接保存为 `bible/story_bible.md`。
